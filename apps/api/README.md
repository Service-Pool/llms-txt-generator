# LLMs.txt Generator API

NestJS-based API для автоматической генерации llms.txt файлов.

## Установка

```bash
npm install
```

## Запуск

```bash
# Development
npm run app:dev

# Production
npm run build
npm run app

# Generation Worker
npm run generation-worker:dev
```

## Миграции

```bash
# Генерация миграции
npm run migration:generate

# Запуск миграций
npm run migration:run

# Откат миграции
npm run migration:revert
```
